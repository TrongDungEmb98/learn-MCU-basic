 Mô tả về Demo

 

Ứng dụng này là một phần của gói phần mềm được phát triển trên mức đỉnh của stack lwIP TCP / IP.
Gói này chứa 9 ứng dụng sau:

  1. Những ứng dụng chạy độc lập (không cần hệ điều hành RTOS):
  • A Webserver. 
  • A TFTP server.
  • A TCP echo client application
  • A TCP echo server application
  • A UDP echo client application
  • A UDP echo server application 
  1. Những ứng dụng chạy trên HDH FreeRTOS:
  • A Webserver based on netconn API.
  • A Webserver based on socket API.
  • A TCP/UDP echo server application based on netconn API. 

 

 

 Thông tin về LwIP

 

lwIP (viết tắt của lightweight IP), là một TCP/IP mã nguồn mở được phát triển bởi Adam Dunkels tại viện Khoa học Máy Tính ở Thụy Điển và ngày nay được duy trì bởi cộng đồng các nhà phát triển trên toàn thế giới.

Những tính năng của lwIP:

- IP (Internet Protocol)
- ICMP (Internet Control Message Protocol)
- UDP (User Datagram Protocol)
- TCP (Transmission Control Protocol)
- API cho su thuc thi nang cao
- Optional Berkeley-alike socket API
- DHCP (Dynamic Host Configuration Protocol)
- PPP (Point-to-Point Protocol)
- ARP (Address Resolution Protocol) cho Ethernet

Tham khảo thêm thông tin tại:  http://savannah.nongnu.org/projects/lwip/
